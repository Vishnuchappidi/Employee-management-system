\# Employee Management System



A Full Stack Employee Management System built using Spring Boot and React.



\## Tech Stack



\### Backend

\- Java

\- Spring Boot

\- Spring Data JPA

\- MySQL



\### Frontend

\- React

\- Vite

\- Axios

\- Bootstrap



\## Features

\- Add Employee

\- View Employees

\- Update Employee

\- Delete Employee
